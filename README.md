### ESM в браузере

- Подключение скрипта через тег
- Вёдра (файлы), защитные контейнеры (модули) и бассейн кода
- Implicit script node

- 'ctrl + ~'
- 'view > terminal'

### Основные полезные команды

- путь (pwd)
- лист (ls)
